A modern, professional, RTL-first WordPress theme for corporate websites with full Farsi/Persian support.

== Description ==

PersiaPro is a production-ready WordPress theme designed primarily for company and corporate websites with Farsi (Persian) as the main language. It features a clean, trustworthy, professional design with elegant Persian aesthetic touches.

= Key Features =

* **RTL-First Design** – Built from the ground up for right-to-left languages with full rtl.css support
* **Fully Customizable** – Extensive WordPress Customizer options (colors, typography, header, hero, layout, footer, blog)
* **Multilingual Ready** – Compatible with WPML, Polylang, and TranslatePress
* **Gutenberg Ready** – Full block editor support with wide and full alignment
* **Page Templates** – Full Width, No Header/Footer, Landing Page
* **WooCommerce Ready** – Basic WooCommerce styling included
* **Performance Optimized** – Clean code, minimal dependencies, Google Fonts with display=swap
* **Child Theme Ready** – Easy to extend with a child theme

= Customizer Panels =

1. **Colors** – Primary, Secondary, Accent, Text, Background, Header/Footer colors
2. **Typography** – Farsi-friendly fonts (Vazirmatn, Vazir, IRANSans), sizes, line heights
3. **Header** – Sticky/transparent header, top bar, social links
4. **Hero / Banner** – Background image, overlay, headline, CTA buttons, height
5. **Layout** – Container width, sidebar position, boxed/full-width, back to top
6. **Footer** – Widget columns (1-4), copyright, social icons
7. **Blog** – Grid/list layout, excerpt length, read more text
8. **Advanced** – Custom CSS, Google Analytics ID

== Installation ==

1. Download `persiapro.zip`
2. Go to WordPress Admin → Appearance → Themes → Add New → Upload Theme
3. Choose the ZIP file and click "Install Now"
4. Activate the theme
5. Go to Appearance → Customize to configure your site

= Recommended Setup =

1. Set site language to Persian (فارسی) in Settings → General
2. Upload your logo in Customize → Site Identity
3. Create a Primary Menu in Appearance → Menus
4. Configure Hero section in Customize → PersiaPro Hero / Banner
5. Add widgets to Footer columns in Appearance → Widgets

== Frequently Asked Questions ==

= Does this theme support RTL? =

Yes! PersiaPro is RTL-first. When WordPress is set to a RTL language (like Persian), the theme automatically loads rtl.css and applies correct text direction.

= Can I use this with Elementor? =

Yes. PersiaPro works well with page builders including Elementor, while also being optimized for the native Gutenberg block editor.

= Is WooCommerce supported? =

Basic WooCommerce styles are included. For advanced shop customization, consider a WooCommerce-specific plugin or child theme.

= How do I create a child theme? =

Create a folder `persiapro-child` with a `style.css` containing:

/*
Theme Name: PersiaPro Child
Template: persiapro
*/

Then add your customizations to the child theme.
